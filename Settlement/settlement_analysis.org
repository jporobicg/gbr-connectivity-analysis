* Methodology

** Statistical Improvements Over Original Approach

*** 1. Binomial Likelihood (Not Aggregated Proportions)

*Original Approach*:
#+BEGIN_SRC python
# Aggregate to proportions, losing sample size information
df_agg['PropCompetent'] = df_agg['NoSet'] / (df_agg['NoSet'] + df_agg['NoNotSet'])
# Fit to proportions - treats n=10 and n=100 equally!
#+END_SRC

*Improved Approach*:
#+BEGIN_SRC python
# Use raw counts directly
neg_log_likelihood = -sum(
    n_settled * log(p_competent) + 
    (n_total - n_settled) * log(1 - p_competent)
)
# Automatically weights by sample size!
#+END_SRC

*** 2. Pool All Treatments (Not Just Rubble)

*Rationale*: For GBR-wide connectivity modeling, we don't know substrate types at each reef. Pooling all treatments:
- Increases sample size 5-150x
- Captures natural variation across substrate types
- More representative of field conditions (multiple substrate types)

*** 3. Overdispersion Quantification

Settlement data shows *overdispersion* (φ > 1): more variability than binomial model predicts.

*Sources of overdispersion*:
- Maternal effects (quality varies by spawning batch)
- Unmeasured environmental factors
- Developmental variation
- Batch effects

*Estimation*:
#+BEGIN_SRC python
# Pearson residuals
expected_var = p * (1 - p) / n
pearson_resid = (observed - predicted) / sqrt(expected_var)

# Dispersion parameter
phi = sum(pearson_resid^2) / (n_obs - n_params)
# phi = 1: binomial (no overdispersion)
# phi > 1: overdispersed
#+END_SRC

*Results*: All families show φ = 4-7, indicating substantial biological variability.

*** 4. Replicate-Level Bootstrap (Not Aggregated)

*Original*: Bootstrap aggregated observations (by age)
*Improved*: Bootstrap individual replicates (wells/plates)

This is more conservative and respects experimental structure.

** Competency Models

Three models tested for each family:

*** Logistic (Sigmoidal)
#+BEGIN_SRC python
competency(age) = L / (1 + exp(-k * (age - x0)))
#+END_SRC

Parameters:
- =L=: Maximum competency (asymptote)
- =k=: Growth rate (steepness of curve)
- =x0=: Inflection point (≈ TC50)

*** Gompertz
#+BEGIN_SRC python
competency(age) = a * exp(-b * exp(-c * age))
#+END_SRC

Parameters:
- =a=: Maximum competency (asymptote)
- =b=: Displacement parameter
- =c=: Growth rate

*** Weibull
#+BEGIN_SRC python
competency(age) = a * (1 - exp(-b * (age - tc)^v))  for age > tc
                = 0                                  for age ≤ tc
#+END_SRC

Parameters:
- =tc=: Precompetency period (onset age)
- =a=: Maximum competency
- =b=: Scale parameter
- =v=: Shape parameter

*** Model Selection
- Use AICc (corrected AIC for small samples)
- AICc = 2*k + 2*NLL + 2*k*(k+1)/(n-k-1)
- Choose model with lowest AICc
- Δ AICc < 2: Models essentially equivalent

import numpy as np

def competency_acroporidae(age_days):
    """Acroporidae competency function (Weibull)"""
    tc, a, b, v = 3.9246, 0.5391, 0.3966, 0.9440
    age = np.atleast_1d(age_days)

competency_acroporidae <- function(age_days) {
  # Weibull model
  tc <- 3.9246
  a <- 0.5391
  b <- 0.3966
  v <- 0.9440
  
  competency <- ifelse(
    age_days > tc,
    a * (1 - exp(-b * (age_days - tc)^v)),

Advantages:
- Incorporate prior knowledge
- Natural uncertainty quantification
- Handle complex models

** Appendix A: Model Equations in Detail

*** Weibull Model (6 families use this)

Full form:
#+BEGIN_EXAMPLE
competency(age) = a * (1 - exp(-b * (age - tc)^v))  if age > tc
                = 0                                  if age ≤ tc
#+END_EXAMPLE

Interpretation:
- *tc*: Precompetency period (no settlement before this age)
- *a*: Maximum competency (asymptotic limit, 0 < a ≤ 1)
- *b*: Scale parameter (controls rate of increase)
- *v*: Shape parameter:
  - v < 1: Decelerating increase (fast initial rise, then slow)
  - v = 1: Exponential increase
  - v > 1: Accelerating increase (slow start, then rapid)

Derivative (rate of competency increase):
#+BEGIN_EXAMPLE
d/dt competency = a * b * v * (age - tc)^(v-1) * exp(-b * (age - tc)^v)
#+END_EXAMPLE

*** Logistic Model (Poritidae uses this)

Full form:
#+BEGIN_EXAMPLE
competency(age) = L / (1 + exp(-k * (age - x0)))
#+END_EXAMPLE

Interpretation:
- *L*: Maximum competency (asymptotic limit)
- *k*: Growth rate (steepness of curve)
- *x0*: Inflection point (age at 50% of maximum, approximately TC50)

Derivative:
#+BEGIN_EXAMPLE
d/dt competency = L * k * exp(-k * (age - x0)) / (1 + exp(-k * (age - x0)))^2
#+END_EXAMPLE

Maximum rate occurs at age = x0

*** Gompertz Model (Not selected for any family)

Full form:
#+BEGIN_EXAMPLE
competency(age) = a * exp(-b * exp(-c * age))
#+END_EXAMPLE

Interpretation:
- *a*: Maximum competency
- *b*: Displacement parameter (shifts curve left/right)
- *c*: Growth rate

** Appendix B: Statistical Formulas

*** Binomial Log-Likelihood

#+BEGIN_EXAMPLE
log L = Σ [n_settled * log(p) + n_not_settled * log(1 - p)]

where:
  p = competency(age | parameters)
  n_settled = number settled at given age
  n_not_settled = number not settled
#+END_EXAMPLE

*** Overdispersion Estimation

#+BEGIN_EXAMPLE
Pearson residual: r = (observed - expected) / sqrt(expected_variance)

Expected variance (binomial): V = n * p * (1 - p)

Dispersion parameter: φ = Σ r^2 / (n_obs - n_params)

Adjusted variance: V_adjusted = φ * V
#+END_EXAMPLE

*** AICc (Corrected Akaike Information Criterion)

#+BEGIN_EXAMPLE
AIC = 2 * k + 2 * NLL

AICc = AIC + (2 * k * (k + 1)) / (n - k - 1)

where:
  k = number of parameters
  NLL = negative log-likelihood
  n = number of observations
#+END_EXAMPLE

*** Bootstrap Confidence Intervals

#+BEGIN_EXAMPLE
# Generate B bootstrap samples
for i in 1:B:
  1. Resample data with replacement
  2. Fit model to bootstrap sample
  3. Store parameters

# Percentile confidence interval
CI_lower = percentile(bootstrap_params, 2.5)
CI_upper = percentile(bootstrap_params, 97.5)
#+END_EXAMPLE
